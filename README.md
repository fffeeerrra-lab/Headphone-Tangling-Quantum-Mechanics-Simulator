## Getting Started

### Prerequisites

- [Clarinet](https://github.com/hirosystems/clarinet) v3.4.0 or higher
- [Node.js](https://nodejs.org/) v16 or higher
- [Git](https://git-scm.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/fffeeerrra-lab/Headphone-Tangling-Quantum-Mechanics-Simulator.git
   cd Headphone-Tangling-Quantum-Mechanics-Simulator
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Check contract syntax:
   ```bash
   clarinet check
   ```

4. Run tests:
   ```bash
   npm test
   ```
